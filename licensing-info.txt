###############################################################################
## Ridgeline
###############################################################################

Homepage: https://github.com/gooberRF/ridgeline

Ridgeline is a modular game patching framework developed by Chris "Goober"
Parsons. The Ridgeline source code is available on GitHub and licensed under
the Mozilla Public License 2.0. See LICENSE.txt in the root of the
distribution.

Ridgeline began as a fork of Alpine Faction (an unrelated patch project for
Red Faction by the same author) but has been re-purposed as a general-purpose
multi-game framework. Its hooking primitives, DLL injection harness, crash
handler, launcher framework, and logging library are inherited from that
lineage. Alpine Faction itself descended from Dash Faction by Rafalh; the
attribution blocks for both projects are preserved below.

Ridgeline uses several open-source third-party components. Their licenses are
listed below.

NOTE TO MAINTAINERS: keep this file in sync with vendor/. When you add a
vendor lib, add a section here. When you remove one, drop its section.
The file is copied next to the launcher exe at install time.
